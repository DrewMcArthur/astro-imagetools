### `objectFit`

**Type:** `"fill" | "contain" | "cover" | "none" | "scale-down"`

**Default:** `"cover"`

The value of the `object-fit` CSS property of the generated `<img />` element.

**Code example:**

```astro
<Image
  src="https://picsum.photos/200/300"
  alt="A random image"
  objectFit="contain"
/>
```
